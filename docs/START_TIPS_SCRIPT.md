# 开唱小抄脚本

## 产品边界

`今晚唱什么` 不控制真实 KTV 设备，不接入第三方音乐平台私有接口，也不播放、下载或复用歌曲素材。它专注在开唱前的歌单准备：把用户主动分享、粘贴或选择的歌单内容，与本地参考 fixture 核对后，按当晚场景和可用的声线来源排成更顺手的顺序。215 首参考数据不代表实时平台曲库或场所可唱情况。

## 30 秒说明

很多聚会点歌的问题不是“没有歌”，而是“大家喜欢听的歌不一定适合唱”。今晚唱什么可以导入常听歌单，逐首核对本地参考匹配，再按朋友局、生日局、车载 K 歌等场景排一份顺手的歌单。每首歌会展示有数据依据的适合理由、注意点和备选歌；精确调性建议只在有效实测音区与已知歌曲音域同时存在时出现。

## 5 分钟体验方式

- 想先看效果：点 `用示例歌单`，看整理歌名和 KTV 匹配结果。
- 手上已有歌单：粘贴公开链接或文本，也可以识别截图，再修正歌名和歌手。
- 临时想开唱：直接进 `排今晚歌单`，选场景、人数、时长和合唱偏好。
- 担心高音吃力：进 `测一下音域`，录 10 秒查看“本次唱到的音区”；赶时间可使用明确标记的“常见音域参考”。
- 要发给朋友：进 `发给朋友` 复制精简歌单、分享详细 `.txt` 文件、保存海报或打开系统分享。
- 到了现场：打开 `开唱小抄`；内容会按朋友、生日、团建、车载、情侣或独自练歌分别给出开始、换歌和协作方式。

## 开唱小抄

- 朋友、生日和团建按实际参与者安排暖场、认领和接唱，不混用角色。
- 情侣局只使用两人轮唱与另一位接歌语义。
- 独自练歌按状态、嗓音负担和复习标记组织，不出现群聊、大家或合唱指令。
- 车载场景始终把安全放在第一位：驾驶者不操作手机，由乘客点歌和切歌；没有乘客时等安全停车后再调整。
